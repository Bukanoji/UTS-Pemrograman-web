Pengalaman Kerja di Cyber Security

Saya memiliki pengalaman dalam keamanan siber, khususnya dalam melakukan pengujian penetrasi (pentesting) untuk berbagai aplikasi dan infrastruktur.

Pengalaman Pentesting

  • Melakukan pengujian penetrasi terhadap aplikasi web dan mobile untuk mengidentifikasi kerentanan.
  • Menilai keamanan jaringan dengan melakukan pengujian intrusi dan analisis risiko.
  • Menyusun laporan hasil pentesting dengan rekomendasi perbaikan untuk meningkatkan keamanan sistem.

Teknologi dan Alat yang Digunakan

Berpengalaman menggunakan alat-alat seperti:

  • Burp Suite
  • Nmap
  • Metasploit
  • OWASP ZAP
  • Wireshark

Sertifikasi

Mendapatkan sertifikasi di bidang keamanan siber, termasuk:

  • Certified Ethical Hacker (CEH)
  • Offensive Security Certified Professional (OSCP)
  • CompTIA Security+
